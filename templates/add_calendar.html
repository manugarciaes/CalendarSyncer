Add Calendar

Add your calendar to sync appointments and availability using an ICS feed URL.

How it works
  1. Copy your calendar's ICS feed URL from your calendar provider
  2. Enter the URL and calendar name below
  3. Set how often you want to refresh calendar data
  4. Click "Add Calendar" to start syncing
Where to find ICS URLs
  • Google Calendar: Calendar Settings → "Secret address in iCal format"
  • Outlook: Calendar Settings → Shared Calendars → Publish → ICS link
  • Apple Calendar: Calendar → Share → Public Calendar → Copy link
  • Other providers: Look for "ICS", "iCal" or "Calendar Feed" options
Must be a valid HTTP or HTTPS URL to an ICS calendar feed
How often to check for calendar updates (recommended: 60 minutes)
